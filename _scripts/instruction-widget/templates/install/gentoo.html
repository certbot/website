Experimental

Currently, the only packaged certbot DNS plugin in the official Gentoo repository is app-crypt/certbot-dns-nsone. The instructions below pertaining the use of an overlay, which is maintained by a member of the Community voluntarily, are experimental and are not officially supported by Gentoo, Let's Encrypt nor EFF.

Bug reporting

If you encounter a problem with or a bug in an ebuild in the `certbot-dns-plugins` overlay, please open an issue on the issues page on the overlays Github repository or open a ticket on Gentoo's Bugzilla. When using Gentoo's Bugzilla, please explicitly mention the overlay using "::certbot-dns-plugins" to ensure that the bug is properly assigned to the repository owner.

All contents of the `certbot-dns-plugins` overlay mentioned in these instructions are provided "AS IS".

  • Enable the `certbot-dns-plugins` overlay

    If you require any DNS plugin other than the app-crypt/certbot-dns-nsone plugin, you will need the required ebuilds to install them. An overlay called certbot-dns-plugins exists with all other certbot DNS plugins. To be able to use those ebuilds, you will need to add the aformentioned overlay.

    Using `eselect`

    If not installed already, install app-eselect/eselect-repository: 

    sudo emerge -av app-eselect/eselect-repository dev-vcs/git

    Next, enable the overlay and sync it: 

    1. sudo eselect repository enable certbot-dns-plugins
      2. 
    
    2. sudo emerge --sync certbot-dns-plugins

    Using `layman`

    If you're using layman for managing your overlays, you can simply run: 

    sudo layman -a certbot-dns-plugins

    Manual overlay installation

    If you don't want to use `eselect-repository` or `layman` to manage your overlays, you can also manually install the overlay. Please see the Gentoo Wiki article about /etc/portage/repos.conf for more information.

    ACCEPT_KEYWORDS

    All ebuilds have the testing KEYWORDS variable set, including potential dependencies from this overlay. Please add the appropriate ACCEPT_KEYWORDS when installing any of the ebuilds.
